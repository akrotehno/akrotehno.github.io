A great thank you to all who have participated in the creation of useful open source tools and we wish you the best. We are grateful for the tools that we use on our website and are free for everyone to use.

NodeJS

All of our services are mainly built on this unique cross-platform Javascript environment. Thanks to NodeJS, we also have easy access to tons of useful open-source packages. NodeJS provides us with the essentials to build our services faster, easier, and better.

Atom

Our favorite tool to edit code for programming various applications. Developed by Github and written in NodeJS.

Github

Github hosts our website freely using “Github-Pages”. We also use Github for collaborating with more developers and versioning our updates.

Hexo

With Hexo we have a strong foundation in developing our website. A framework to generate static files, offering amazing performance and ability to host on Github. Hexo is built on NodeJS.

Openshift

Openshift provides us with rapid deployment of our NodeJS backend services. Thanks to their free plan, we have been able to run and test our applications before scaling up without limit of time.

Podio

We use Podio for workplace collaboration, project management, and thanks to their API, we are able to exchange information through our services.

JQuery

A helpful Javascript Library to write less and do more with our web applications. Thanks to Jquery we also have access to many external tools that enhance our user experience.

Gimp

Almost every graphic here is done with this great tool. Used for photo editing, creating graphics, image batch processing, etc.
