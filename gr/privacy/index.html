1. Τι κάνουμε με τα στοιχεία σας

Όταν αγοράζετε κάτι από εμάς, ως μέρος της διαδικασίας αγοράς και πώλησης, συλλέγουμε τις προσωπικές πληροφορίες που μας δίνετε, όπως το όνομά σας, τη διεύθυνση και διεύθυνση ηλεκτρονικού ταχυδρομείου.

Κατά την περιήγησή σας στην ιστοσελίδα μας, επίσης λαμβάνουμε αυτόματα την IP (Internet Protocol) διεύθυνση του υπολογιστή σας προκειμένου να μας δώσετε πληροφορίες που μας βοηθά να μάθουμε για το πρόγραμμα περιήγησης σας και το λειτουργικό σας σύστημα.

Με την άδειά σας, μπορούμε να σας στείλουμε ηλεκτρονικά μηνύματα για τις δραστηριότητές μας, τα νέα προϊόντα και άλλες ενημερώσεις.

2. Πώς παίρνουμε τη συγκατάθεσή σας

Όταν μας παρέχετε με προσωπικές πληροφορίες για να ολοκληρώσετε μια συναλλαγή, για βεβαίωση της πιστωτικής σας κάρτα, για παραγγελία, για να μεριμνήσετε μια παράδοση ή επιστροφή αγοράς, σημαίνει ότι δίνετε τη συγκατάθεσή σας για την συλλογή και τη χρήση αυτών των πληροφοριών για μόνο αυτό το συγκεκριμένο λόγο.

Αν ζητάμε τα προσωπικά σας στοιχεία για ένα δευτερεύων λόγο, όπως το μάρκετινγκ, είτε θα σας ρωτήσουμε ευθέως για ρητή συγκατάθεσή σας, ή θα σας δώσουμε την ευκαιρία να πείτε όχι.

3. Πώς μπορείτε να αποσύρετε τη συγκατάθεσή σας

Αν μετά το opt-in, αλλάξετε γνώμη, μπορείτε να ανακαλέσετε τη συγκατάθεσή σας για να επικοινωνούμε μαζί σας, για τη συνεχή συλλογή, χρήση ή αποκάλυψη των πληροφοριών σας, ανά πάσα στιγμή, επικοινωνώντας μαζί μας στο support@akrotehno.com

4. Γνωστοποίηση

Μπορούμε να αποκαλύψουμε τα προσωπικά σας στοιχεία, αν είμαστε υποχρεωμένοι από το νόμο ή εάν παραβιάσετε τους Όρους Παροχής Υπηρεσιών μας.

5. Υπηρεσίες Τρίτων

Σε γενικές γραμμές, οι πάροχοι τρίτων που χρησιμοποιούνται από εμάς θα συλλέγουν, χρησιμοποιούν και αποκαλύπτουν τις πληροφορίες σας στο βαθμό που είναι αναγκαίο για να τους επιτρέψει να εκτελούν τις υπηρεσίες που μας παρέχουν.

Ωστόσο, ορισμένοι φορείς παροχής υπηρεσιών από τρίτους, όπως οι πύλες πληρωμής και άλλους επεξεργαστές πληρωμής, έχουν τις δικές τους πολιτικές απορρήτου σε σχέση με τις πληροφορίες που απαιτούνται για την παροχή σε αυτούς για συναλλαγές αγορών που σχετίζονται με εσάς.

Για αυτούς τους παρόχους, σας συνιστούμε να διαβάσετε τις πολιτικές απορρήτου τους, ώστε να μπορείτε να καταλάβετε τον τρόπο με τον οποίο τα προσωπικά σας στοιχεία θα χρησιμοποιούνται από αυτούς τους προμηθευτές.

Ειδικότερα, θυμηθείτε ότι ορισμένοι πάροχοι μπορούν να βρίσκονται εντός ή διαθέτουν εγκαταστάσεις που βρίσκονται σε διαφορετική δικαιοδοσία, είτε από εσάς ή από εμάς. Έτσι, αν έχετε επιλέξει να προχωρήσετε σε συναλλαγή που περιλαμβάνει τις υπηρεσίες του παρόχου υπηρεσιών από τρίτους, τότε οι πληροφορίες σας μπορεί να υπόκεινται στους νόμους της δικαιοδοσίας (ες) στις οποίες ο πάροχος υπηρεσιών ή οι εγκαταστάσεις του βρίσκονται.

Μόλις φύγετε από την ιστοσελίδα μας ή κατευθυνθείτε σε μια ιστοσελίδα ή εφαρμογή τρίτων, δεν είστε πλέον στην παρούσα Πολιτική Απορρήτου ή Όροι Παροχής Υπηρεσιών της ιστοσελίδας μας.

6. Ασφάλεια

Για να προστατέψουμε τα προσωπικά σας στοιχεία, αναλαμβάνουμε λογικές προφυλάξεις και ακολουθούμε τις βέλτιστες πρακτικές για να βεβαιωθούμε ότι δεν θα χαθούν ή θα υπόκεινται σε κατάχρηση, πρόσβαση, αποκάλυψη, αλλοίωση ή καταστροφή, ανάρμοστα.

Αν μας δώσετε τα στοιχεία της πιστωτικής σας κάρτας, οι πληροφορίες κρυπτογραφούνται χρησιμοποιώντας ασφαλή τεχνολογία (SSL). Αν και καμία μέθοδος της μετάδοσης μέσω του Διαδικτύου ή ηλεκτρονικής αποθήκευσης είναι 100% ασφαλής, θα ακολουθούν γενικά αποδεκτά πρότυπα.

7. Ενηλικιότητα

Με τη χρήση αυτής της ιστοσελίδας, δηλώνετε ότι είστε τουλάχιστον στην ηλικία ενηλικίωσης στο κράτος ή την επαρχία της κατοικίας σας, ή ότι είστε στην ηλικία της πλειοψηφίας στην πολιτεία ή επαρχία της διαμονής σας και μας έχετε δώσει τη συγκατάθεσή σας για να επιτρέψουμε σε μικρότερα εξαρτώμενα σας να χρησιμοποιούν αυτή την ιστοσελίδα.

8. Αλλαγή στην παρούσα Πολιτική

Διατηρούμε το δικαίωμα να τροποποιήσουμε αυτήν την πολιτική ανά πάσα στιγμή, έτσι παρακαλούμε να διαβάζετε αυτή τη σελίδα συχνά. Αλλαγές και διευκρινίσεις θα τεθούν σε ισχύ αμέσως μετά την ανάρτησή τους στην ιστοσελίδα. Αν κάνουμε σημαντικές αλλαγές σε αυτή την πολιτική, θα σας ειδοποιήσουμε εδώ ότι έχει ενημερωθεί, έτσι ώστε να είστε ενήμεροι για τις πληροφορίες που συλλέγουμε, πώς τις χρησιμοποιούμε, και κάτω από ποιες συνθήκες, αν υπάρχουν, χρησιμοποιούμε ή / και αποκαλύπτουμε.

Αν η ιστοσελίδα μας έχει αποκτηθεί ή συγχωνευθεί με άλλη εταιρεία, οι πληροφορίες σας μπορεί να μεταφερθούν στους νέους ιδιοκτήτες, ώστε να μπορούν να συνεχιστούν οι υπηρεσίες μας για εσάς.

9. Ερωτήσεις και Στοιχεία Επικοινωνίας

Αν θέλετε: πρόσβαση, διόρθωση, τροποποίηση ή διαγραφή οποιασδήποτε προσωπικής πληροφορίας που έχουμε για σας, αν θέλετε να δηλώσετε παράπονο, ή αναζειτάτε περισσότερες πληροφορίες, επικοινωνήστε μαζί μας μέσω email στο support@akrotehno.com
