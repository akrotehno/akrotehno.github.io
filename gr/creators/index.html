Δημιουργοί ιστοσελίδας

Η κατασκευή της ιστοσελίδας πραγματοποιήθηκε με ιδιαίτερη προσοχή στην ευχρηστία καθώς και στην εμφάνιση. Με αυτό τον τρόπο είμαστε σε θέση να διαδώσουμε αληθινή και άξια πληροφορία σε όλο τον κόσμο.

Το βασικότερο στοιχείο της ιστοσελίδας είναι να αποκτήσουμε μια άριστη καλλιτεχνικο-δημιουργική ζωή μεταξύ μας. Παράλληλα να έχουμε τη δυνατότητα για άμεση επικοινωνία μεταξύ όλων των ανθρώπων των οποίων ταυτίζουμε και έχουμε τα ίδια οράματα και σκοπούς.

Σχεδιασμός και Ανάπτυξη

Ραφαήλ Βαρώνος

Προγραμματισμός, Γραφιστική, Συγγραφή

Λιούμπομιρ Μπούροβ

Επιμέλεια, Συγγραφή

Ένα μεγάλο ευχαριστώ σε όλους που έχουν συμμετάσχει στην κατασκευή χρήσιμων εργαλείων ανοιχτού κώδικου και ευχόμαστε την καλύτερη τους συνέχεια. Είμαστε ευγνώμων για τα παρακάτω εργαλεία όπου χρησιμοποιούμε στην ιστοσελίδα μας και προσφέρονται δωρεάν σε όλους.

NodeJS

Όλες οι υπηρεσίες μας είναι χτισμένες κυρίως σε αυτό το μοναδικό Javascript περιβάλλον. Χάρη στο NodeJS, έχουμε επίσης εύκολη πρόσβαση σε πολλά χρήσιμα πακέτα ανοιχτού κώδικα. Το NodeJS μας παρέχει τα απαραίτητα για την κατασκευή των υπηρεσιών μας γρηγορότερα, ευκολότερα και καλύτερα.

Atom

Το αγαπημένο μας εργαλείο στην επεξεργασία κώδικα για τον προγραμματισμό διάφορων εφαρμογών. Αναπτύχθηκε από Github και γραμμένο σε NodeJS.

Github

To Github φιλοξενεί την ιστοσελίδα μας χρησιμοποιώντας ελεύθερα την υπηρεσία τους, "Github-Pages". Επίσης, χρησιμοποιούμε το Github για τη συνεργασία μας με περισσότερους προγραμματιστές και για να κρατάμε ιστορικό στις εκδόσεις μας.

Hexo

Με το Hexo έχουμε ένα ισχυρό θεμέλιο στην ανάπτυξη της ιστοσελίδας μας. Ένα Framework για τη δημιουργία στατικών αρχείων, προσφέροντας εκπληκτικές επιδόσεις και ικανότητα να φιλοξενείται από το Github. To Hexo είναι χτισμένο σε NodeJS.

Openshift

Το Openshift μας παρέχει ταχεία διάδοση των NodeJS παρασκηνίων υπηρεσιών μας. Χάρη στο ελεύθερο πρόγραμμά τους, είμαστε σε θέση να τρέχουμε και να δοκιμάζουμε τις εφαρμογές μας πριν από την κλιμάκωση τους χωρίς χρονικό περιορισμό.

Podio

Χρησιμοποιούμε το Podio για συνεργασία στο χώρο εργασίας, τη διαχείριση του έργου, και χάρη στο API τους, είμαστε σε θέση να ανταλλάζουμε πληροφορίες μέσω των υπηρεσιών μας.

JQuery

Μια χρήσιμη βιβλιοθήκη Javascript για να γράφουμε λιγότερο κώδικα και να κάνουμε περισσότερα με τις διαδικτυακές εφαρμογές μας. Χάρη στην JQuery έχουμε επίσης πρόσβαση σε πολλά εξωτερικά εργαλεία που ενισχύουν την εμπειρία των χρηστών μας.

Gimp

Σχεδόν κάθε γραφικό εδώ γίνεται με αυτό το εξαιρετικό εργαλείο. Χρησιμοποιείται για την επεξεργασία φωτογραφιών, δημιουργία γραφικών, μαζική επεξεργασία εικόνας, κλπ.
